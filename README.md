# 북돋움 관리대장 검색 프로그램 
**초보 엄마아빠·임신부 지원 사업 [북돋움 책선물]** 사업에서, 이전 연도에 이미 신청한 이용자를 걸러내기 위한 검색 및 필터 프로그램입니다.
본 프로그램은 작년에 지원한 신청자 명단(엑셀)과 새로운 신청자 데이터를 비교하여, 중복 신청을 사전에 차단할 수 있도록 돕습니다.

---

## 주요 기능

###  1. 검색 기능
- 부모이름, 생년월일, 주소, 연락처 등 다양한 항목으로 검색 가능  
- Enter 키로 바로 검색 실행
- 생년월일 자동 자리수 보정 (예: 41218 -> 041218)
- 전화번호 자동 정리 기능 포함 (예: 01012345678 → 010-1234-5678)
- 부분 검색 가능 (대소문자 무시)
###  2. 단일 엑셀 파일 기반
- `북돋움관리대장.xlsx` 파일 **1개만** 사용  
- 파일이 커져도 즉시 읽어와서 검색 가능
###  3. 엑셀 파일 다중 시트 자동 로딩
- 2024년, 2025년, 2026년 형식의 시트를 자동 인식
- 연도 시트가 추가되어도 코드 수정 없이 자동 대응
###  4. 실행 속도 개선
- 엑셀 로딩·파일 감시·검색을 **별도 스레드 처리**  
- exe로 변환해도 켜지는 속도 최대한 빠르게 설계됨
### 5. 안정성 강화
- 빈 시트 자동 무시
- 컬럼 타입 오류 방어
- 엑셀 파일 열림/저장 충돌 방지 구조

## 사용 방법

### 1. 파일 구조
```
📁 폴더/
├── books.py                    # 메인 프로그램
├── 북돋움관리대장.xlsx          # 이전 신청자 명단 (사용자 준비)
└── README.md
```
### 2. 환경 준비
- Python 3.9 이상
- 필요한 패키지 설치: python search_program.py

### 3. 실행 방법 
#### Python
pip install pandas
python books.py

#### exe 파일로 빌드하기
1. PyInstaller 설치
```pip install pyinstaller```
2. exe 생성
```pyinstaller --onefile --noconsole books.py```
3. 결과물
```
dist
└── books.exe
```
books.exe와 북돋움관리대장.xlsx를 같은 폴더에 두고 실행
```
